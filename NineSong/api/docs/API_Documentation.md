## 元数据同步 API

### 支持的歌词格式
系统支持以下歌词文件格式：
- **LRC**: 标准歌词格式（.lrc）
- **TXT**: 纯文本歌词格式（.txt）
- **KRC**: 酷狗音乐歌词格式（.krc）
- **QRC**: QQ音乐歌词格式（.qrc）

### 支持的封面格式
- **JPEG**: JPEG 图片格式（.jpg, .jpeg）
- **PNG**: PNG 图片格式（.png）

### 歌词文件处理说明
- 系统会自动检测音频文件同目录下的歌词文件（支持所有上述格式）
- 优先匹配与音频文件同名的歌词文件
- 支持从音频文件内嵌标签中提取歌词
- 歌词文件会与音频文件一起同步上传和下载

### 歌词格式详细说明

#### LRC格式 (.lrc)
标准歌词格式，支持时间轴同步。格式示例：
```
[00:12.00]第一句歌词
[00:15.30]第二句歌词
[00:18.50]第三句歌词
```

#### TXT格式 (.txt)
纯文本歌词格式，不包含时间轴信息。格式示例：
```
第一句歌词
第二句歌词
第三句歌词
```

#### KRC格式 (.krc)
酷狗音乐专用歌词格式，支持逐字时间轴和特效。格式为二进制编码，需要专门的解析器。

#### QRC格式 (.qrc)
QQ音乐专用歌词格式，支持逐字时间轴和特效。格式为二进制编码，需要专门的解析器。

### 歌词文件自动检测机制

系统在扫描音频文件时会自动检测同目录下的歌词文件：

1. **文件名匹配规则**：
   - 音频文件：`song.mp3`
   - 支持的歌词文件：
     - `song.lrc` (优先)
     - `song.txt`
     - `song.krc`
     - `song.qrc`

2. **检测优先级**：
   - 首先检查 LRC 格式
   - 然后检查 TXT 格式
   - 接着检查 KRC 格式
   - 最后检查 QRC 格式
   - 找到第一个匹配的文件即停止搜索

3. **CUE文件场景**：
   - 对于CUE文件，系统会扫描CUE文件所在目录
   - 检测所有支持的歌词格式文件
   - 自动关联到对应的音频轨道

### 歌词文件上传下载

歌词文件会随音频文件自动同步，无需单独上传。系统会在以下情况自动处理：

1. **上传音频文件时**：
   - 自动检测同目录下的歌词文件
   - 将歌词文件复制到元数据存储目录
   - 在数据库中建立关联关系

2. **下载音频文件时**：
   - 自动下载关联的歌词文件
   - 保持原始文件名和格式

### 歌词文件存储路径

歌词文件存储在以下路径结构：
```
./uploads/metadata/lyrics/{media_file_id}/lyrics.{format}
```

其中 `{format}` 为文件格式扩展名（lrc, txt, krc, qrc）。
