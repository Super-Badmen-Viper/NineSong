# Public Status Notice

This document preserves the status declaration that previously appeared at the top of the public README.

## Archived Public Positioning

According to the archived public README snapshot, the public-facing decisions were:

1. The NineSong backend repository would stop receiving open-source updates and remain frozen at its current public commit.
2. DockerHub distribution would remain free for all users.
3. React-based frontend apps for music, gallery, video, notes, and related scenarios would continue as open-source and free applications, with rollout expected to start around April 2026.
4. The broader multi-scenario data-sharing architecture and AI-related expansion plans would continue through refactoring and further development.
5. A NineSong-Desktop installer flow for Windows, Linux, and macOS, together with a more visual NineSong-Panel deployment experience, was described as a future-facing direction.
6. The Flutter-based NSMusicS client line was described as the cross-platform premium client planned for Android, iOS, Windows, and macOS, with store distribution across Google Play, App Store, and Microsoft Store.

## Interpretation Note

This repository should now be read as an archived public snapshot rather than as the actively advancing upstream for every part of the ecosystem.
