# 公开状态说明

这个文档用于保留旧版公开 README 顶部的状态声明。

## 归档下来的公开定位

按归档下来的公开 README，原有公开表述主要包括：

1. NineSong 后端公开仓库将停止开源更新，并永久停留在当前公开提交。
2. DockerHub 分发会继续保持免费。
3. 面向音乐、相册、影视、笔记等场景的 React 前端应用，会继续以开源免费应用方向推进，预计会在 2026 年 4 月左右开始陆续推出。
4. 更广义的多场景数据共享架构和 AI 相关扩展，仍会继续沿着重构和后续开发推进。
5. 面向 Windows、Linux、macOS 的 NineSong-Desktop 安装导向，以及更可视化的 NineSong-Panel 部署体验，被描述为后续方向之一。
6. 基于 Flutter 的 NSMusicS 客户端产品线，被描述为面向 Android、iOS、Windows、macOS 的跨平台付费客户端，并计划覆盖 Google Play、App Store、Microsoft Store。

## 解释说明

现在更适合把这个仓库理解为归档下来的公开快照，而不是整个生态所有部分仍在持续推进的唯一主仓库。
